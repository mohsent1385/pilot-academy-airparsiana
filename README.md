# وبسایت معرفی رشته خلبانی - ایرپارسیانا

این پروژه یک وبسایت آموزشی و معرفی رشته خلبانی است که با HTML, CSS, JavaScript و Bootstrap ساخته شده است.  
هدف آن آموزش اصول طراحی وب و ارائه اطلاعات کامل درباره مسیر شغلی خلبانی است.

---

## صفحات پروژه
- `index.html` : صفحه اصلی، معرفی رشته خلبانی و مزایای شغلی
- `about.html` : درباره رشته خلبانی، ویژگی‌ها و آینده شغلی
- `blog.html` : لیست مقالات خلبانی
- `single-blog.html` : مقاله تکی و توضیحات کامل
- `contact.html` : فرم مشاوره و تماس با اعتبارسنجی JavaScript

---

## تکنولوژی‌ها
- HTML5
- CSS3
- JavaScript
- Bootstrap 5 (RTL)
